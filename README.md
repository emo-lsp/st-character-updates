# st-character-updates

SillyTavern 角色卡自动更新资源仓库

Auto-update resources for SillyTavern character cards

---

本仓库用于托管 SillyTavern 角色卡的更新文件（角色卡 PNG + 更新日志），配合[酒馆助手](https://github.com/StageDog/tavern_resource)的自动更新脚本使用。

This repository hosts SillyTavern character card update files (card PNG + changelog), used with [Tavern Helper](https://github.com/StageDog/tavern_resource)'s auto-update script.
